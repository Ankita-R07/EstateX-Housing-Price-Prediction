# 🏠 EstateX: Housing Price Prediction System

## 📖 Project Overview
EstateX is a machine learning-based system designed to predict housing prices using real estate data. The project analyzes various factors such as location, property type, and furnishing to estimate property prices and provide valuable insights for buyers, sellers, and investors.

---

## 🎯 Objectives
- Predict housing prices using machine learning models  
- Perform Exploratory Data Analysis (EDA) to understand pricing trends  
- Identify key factors influencing property prices  
- Compare different regression models for performance  
- Support data-driven decision-making in the real estate sector  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
- **Techniques:** Regression, Machine Learning, Data Analysis  

---

## ⚙️ Methodology
1. Data Collection (real estate dataset ~3L records)  
2. Data Cleaning and Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Model Building (Linear Regression, Random Forest)  
6. Model Evaluation (MAE, RMSE, R² Score)  
7. Visualization of insights  

---

## 📊 Results
- Processed and analyzed 3L+ real estate records  
- Random Forest outperformed Linear Regression  
- Achieved R² score of ~0.23  
- Identified location, furnishing, and property type as key factors  

---

## 🌍 Impact
- Helps buyers and investors estimate fair property prices  
- Supports real estate businesses with data-driven insights  
- Improves transparency in property valuation  

---

## 🚀 Future Scope
